# 载具夹取机构选型

## 机构概述

载具夹取机构用于夹持工件或载具，进行搬运、定位或固定。特点是侧向夹持、依靠摩擦力保持、常需要多点夹取。

### 典型应用场景

- 搬运机械手夹取
- 工件定位夹紧
- 载具临时固定
- 多工位夹持转移

---

## ⚠️ 首要步骤：确认夹持结构

**在开始计算前，必须先向用户确认夹持结构：**

> **是否使用卡钩（钩住产品凹槽）？**

这个确认至关重要，因为两种结构的计算方法完全不同：

| 夹持结构 | 保持原理 | 正压力来源 | 所需气缸力 |
|----------|----------|------------|------------|
| **卡钩式** | 机械钩住凹槽 | 产品自重压在卡钩上 | 较小 |
| **摩擦式** | 摩擦力夹紧 | 气缸夹持力 | 较大 |

---

## 一、卡钩式夹持选型

### 结构特点

```
卡钩受力示意（水平搬运）：

    夹爪
     │
     ▼
   ┌───┐
   │卡钩│←───── F_friction（摩擦力，阻碍脱离）
   └─┬─┘
     │ 钩住
  ───┴─── 产品凹槽
     ↓
   G = mg（产品重量）
```

**关键理解：**
- 产品重量向下 → 压在卡钩上 → 产生正压力
- 正压力 N = 产品重量 = m × g
- 气缸只需提供：**卡钩脱离凹槽的力**
- 脱离阻力 = 卡钩与产品之间的摩擦力

### 计算公式

```
正压力：N = m × g

脱离阻力（摩擦力）：F_friction = μ × N = μ × m × g

所需气缸力：F_gripper = F_friction × K = μ × m × g × K
```

| 参数 | 说明 | 典型值 |
|------|------|--------|
| m | 产品+夹爪总质量 | kg |
| g | 重力加速度 | 9.81 m/s² |
| μ | 卡钩与产品摩擦系数 | 0.2~0.4（金属-金属）|
| K | 安全系数 | 1.5~2.0 |

### 计算示例

**工况：**
- 产品+夹爪质量：1 kg
- 卡钩材质：金属
- 产品凹槽材质：金属
- 使用压力：0.5 MPa

**计算：**
```
N = 1 × 9.81 = 9.81 N（正压力）
μ = 0.3（金属-金属）
F_friction = 0.3 × 9.81 = 2.94 N（脱离阻力）
K = 2.0（安全系数）

F_gripper = 2.94 × 2.0 = 5.88 N
```

**选型：** HFZ16（夹持力32N，余量5.4倍，完全满足）

## 二、摩擦式夹持选型

### 基本夹持力计算

```
F_clamp = m × g × K_safety / (n × μ)
```

| 参数 | 说明 | 单位 |
|------|------|------|
| m | 工件质量 | kg |
| g | 重力加速度 9.81 | m/s² |
| K_safety | 安全系数 | - |
| n | 夹持点数量 | - |
| μ | 摩擦系数 | - |

### 不同工况的夹持力

**水平夹持，垂直搬运：**
```
F_clamp = m × g × K_safety / (2 × μ)
```
（假设2点夹持）

**水平夹持，水平加速：**
```
F_clamp = m × a × K_safety / (2 × μ)
```
- a: 加速度 (m/s²)

**倾斜夹持：**
```
F_clamp = m × g × K_safety / (n × μ × cos(θ))
```
- θ: 夹持面与垂直方向夹角

### 夹臂力矩计算

对于夹臂式机构：

```
F_gripper × L1 = F_clamp × L2
F_clamp = F_gripper × (L1 / L2)
```

- L1: 气缸到支点距离
- L2: 夹持点到支点距离
- L1/L2: 力臂比

### 摩擦系数选择

| 夹爪材料 | 工件材料 | 摩擦系数 μ |
|----------|----------|------------|
| 橡胶 | 金属 | 0.5~0.8 |
| 橡胶 | 塑料 | 0.4~0.6 |
| 金属 | 金属 | 0.2~0.4 |
| 金属 | 塑料 | 0.2~0.3 |
| 齿纹金属 | 金属 | 0.4~0.6 |

## 选型步骤

### Step 1: 收集参数

- 工件质量 m (kg)
- 工件尺寸 (mm)
- 夹持点数量 n
- 夹持方式（外夹/内撑）
- 运动方向（水平/垂直/倾斜）
- 加速度 a (m/s²)
- 夹爪材料/摩擦系数 μ

### Step 2: 计算所需夹持力

```
F_clamp = m × g × K_safety / (n × μ)
```

安全系数选择：
- 平稳搬运：1.5~2.0
- 快速搬运：2.0~3.0
- 高加速度：3.0~4.0

### Step 3: 计算气缸出力

**直接夹持：**
```
F_gripper = F_clamp
```

**夹臂机构：**
```
F_gripper = F_clamp × (L2 / L1)
```

### Step 4: 计算缸径

```
D = √(4 × F_gripper × K / (π × P × η))
```

### Step 5: 选择行程

开闭行程：
```
S = (W_open - W_close) / 2 × (L1 / L2)
```
- W_open: 开放时夹持点间距
- W_close: 夹紧时夹持点间距

### Step 6: 选择气缸类型

- 平行夹：简单，通用
- 角度夹：力臂可变
- 旋转夹：90°旋转+夹紧

## 典型配置示例

### 示例1：小型工件夹取

**工况：**
- 工件质量：0.2 kg
- 夹持方式：外夹
- 夹持点：2点
- 夹爪：橡胶
- 搬运方向：垂直

**计算：**
```
μ = 0.6 (橡胶-金属)
K_safety = 2.0

F_clamp = 0.2 × 9.81 × 2.0 / (2 × 0.6)
        = 3.924 / 1.2
        = 3.27 N

考虑夹臂力臂比 L2/L1 = 1.5
F_gripper = 3.27 × 1.5 = 4.9 N
```

**选型结果：**
- 气缸：HD10 或 MHZ2-10D
- 夹持力：>5N
- 行程：10~20mm

### 示例2：中型工件夹取

**工况：**
- 工件质量：2 kg
- 夹持方式：外夹
- 夹持点：2点
- 夹爪：齿纹金属
- 搬运方向：垂直+水平加速
- 加速度：5 m/s²

**计算：**
```
垂直负载：F_v = 2 × 9.81 = 19.6 N
水平负载：F_h = 2 × 5 = 10 N
总负载：F_total = √(19.6² + 10²) = 22 N

μ = 0.5 (齿纹金属-金属)
K_safety = 2.5

F_clamp = 22 × 2.5 / (2 × 0.5)
        = 55 N

夹臂比 L2/L1 = 1.2
F_gripper = 55 × 1.2 = 66 N
```

**选型结果：**
- 气缸：HD20 或 MHZ2-20D
- 夹持力：>80N
- 行程：20~30mm

### 示例3：载具夹紧固定

**工况：**
- 载具质量：5 kg
- 夹持方式：侧面内撑
- 夹持点：4点
- 夹爪：橡胶
- 目的：工序中固定

**计算：**
```
μ = 0.6 (橡胶-金属)
K_safety = 1.5 (静态保持)

F_clamp = 5 × 9.81 × 1.5 / (4 × 0.6)
        = 73.6 / 2.4
        = 30.7 N

内撑力需求更大，取2倍
F_gripper = 30.7 × 2 = 61.4 N
```

**选型结果：**
- 气缸：SC20×30 或 CQ2B20-30D（自制夹爪）
- 数量：4个（或2个联动）
- 单缸出力：>80N

## 夹爪设计要点

### 夹爪形状

| 形状 | 适用场景 | 特点 |
|------|----------|------|
| V型 | 圆柱工件 | 自对中 |
| 平面 | 方形工件 | 简单通用 |
| 齿纹 | 防滑要求高 | 摩擦力大 |
| 橡胶垫 | 易损工件 | 保护表面 |
| 定制 | 异形工件 | 按工件设计 |

### 夹持稳定性

1. **夹持点位置**
   - 尽量靠近重心
   - 对称分布
   - 避免单点悬臂

2. **夹持面积**
   - 足够大以分散压力
   - 避免局部变形

3. **夹持深度**
   - 至少夹持工件高度的1/3
   - 防止翻转

### 柔顺设计

- 弹簧补偿：适应工件尺寸变化
- 浮动机构：补偿定位误差
- 力控：可调节夹持力

## 多点夹取设计

### 同步要求

- 机械同步：齿轮齿条/连杆
- 气路同步：分流阀
- 电气同步：多气缸独立控制

### 布局原则

| 夹取点数 | 布局方式 |
|----------|----------|
| 2点 | 对称分布 |
| 3点 | 三角形分布 |
| 4点 | 矩形四角 |

### 气缸选择

- 优先选用成品气动夹爪
- 大型/异形件用普通气缸自制
- 需同步时选多缸联动机构

## 常见问题与解决

| 问题 | 可能原因 | 解决方案 |
|------|----------|----------|
| 夹持不稳 | 夹持力不足 | 增大缸径/增加夹持点 |
| 工件变形 | 夹持力过大 | 减小缸径/加软垫 |
| 夹持位置偏 | 夹爪对中差 | 调整夹爪/加定位销 |
| 工件滑落 | 摩擦系数低 | 更换夹爪材料/增大夹持力 |
| 开闭不同步 | 气缸速度不一致 | 加节流阀/改机械同步 |
| 夹爪磨损 | 硬度不够 | 淬火/更换材料 |

## 两种夹持方式对比总结

| 对比项 | 卡钩式 | 摩擦式 |
|--------|--------|--------|
| **保持原理** | 机械钩住凹槽 | 摩擦力夹紧 |
| **正压力来源** | 产品自重压在卡钩上 | 气缸夹持力 |
| **计算公式** | F = μ × m × g × K | F = m × g × K / (n × μ) |
| **所需气缸力** | 较小 | 较大 |
| **1kg负载示例** | 约6 N | 约33 N |
| **适用场景** | 产品有凹槽/孔位 | 产品表面平整 |
| **可靠性** | 高（机械限位） | 依赖摩擦系数 |
