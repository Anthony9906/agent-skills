# 气缸选型计算公式

## 基础理论

### 气缸理论出力

**推力（伸出方向）：**
```
F_push = P × (π × D² / 4) × η
```

**拉力（缩回方向）：**
```
F_pull = P × (π × (D² - d²) / 4) × η
```

参数说明：
- P: 使用压力 (MPa)
- D: 缸径 (mm)
- d: 活塞杆直径 (mm)
- η: 效率系数，通常取0.7~0.9

### 活塞杆直径与缸径关系

| 缸径 D (mm) | 活塞杆 d (mm) | d/D 比值 |
|-------------|---------------|----------|
| 10 | 4 | 0.40 |
| 12 | 6 | 0.50 |
| 16 | 6 | 0.38 |
| 20 | 8 | 0.40 |
| 25 | 10 | 0.40 |
| 32 | 12 | 0.38 |
| 40 | 16 | 0.40 |
| 50 | 20 | 0.40 |
| 63 | 20 | 0.32 |
| 80 | 25 | 0.31 |
| 100 | 30 | 0.30 |

## 缸径计算

### 基本公式

**水平运动：**
```
D = √(4 × F × K / (π × P × η))
```

**垂直运动（向上）：**
```
D = √(4 × (m × g + F_friction) × K / (π × P × η))
```

**垂直运动（向下）：**
```
D = √(4 × (m × g - F_friction) × K / (π × P × η))
```

### 标准缸径系列

常用标准缸径：10, 12, 16, 20, 25, 32, 40, 50, 63, 80, 100, 125

计算后向上取整至最近的标准缸径。

## 速度计算

### 理论速度

```
v = Q / A
```
- Q: 空气流量 (L/min)
- A: 活塞面积 (cm²)
- v: 速度 (m/min)

### 实际速度

```
v_actual = v_theory × 0.6~0.9
```

### 速度调节

通过节流阀控制，一般范围：
- 微型气缸：50~500 mm/s
- 小型气缸：50~750 mm/s
- 中型气缸：50~1000 mm/s

推荐工作速度：100~500 mm/s

## 冲击力计算

### 挡停冲击力

```
F_impact = m × v² / (2 × S_buffer) × K_dynamic
```

参数：
- m: 载具+工件质量 (kg)
- v: 载具撞击速度 (m/s)
- S_buffer: 缓冲行程 (m)
- K_dynamic: 动载系数，取1.5~2.5

### 能量吸收

```
E = 0.5 × m × v²
```

缓冲器吸收能量需大于此值。

## 摩擦力计算

### 滑动摩擦

```
F_friction = μ × N
```

常用摩擦系数：
| 接触材料 | 静摩擦系数 μs | 动摩擦系数 μd |
|----------|---------------|---------------|
| 钢/钢 | 0.15~0.30 | 0.10~0.20 |
| 钢/铝 | 0.20~0.35 | 0.15~0.25 |
| 钢/工程塑料 | 0.25~0.40 | 0.20~0.30 |
| 铝/铝 | 0.20~0.35 | 0.15~0.25 |

## 夹持力计算

### 基本夹持力

```
F_clamp = m × g × K / (n × μ × cos(θ))
```

参数：
- m: 工件质量 (kg)
- n: 夹持点数量
- μ: 摩擦系数
- θ: 夹持角度（垂直夹持时θ=0）

### 力矩放大

夹臂式机构：
```
F_gripper × L1 = F_clamp × L2
F_clamp = F_gripper × (L1 / L2)
```

## 偏心负载

### 允许偏心量

```
e_max = M_allow / F
```

- M_allow: 气缸允许力矩 (N·m)
- F: 实际出力 (N)
- e: 偏心量 (m)

### 偏心力矩

```
M = F × e
```

需满足：M < M_allow

### 常见气缸允许力矩

| 缸径(mm) | 允许弯曲力矩(N·m) | 允许扭转力矩(N·m) |
|----------|-------------------|-------------------|
| 16 | 0.3 | 0.15 |
| 20 | 0.6 | 0.3 |
| 25 | 1.2 | 0.6 |
| 32 | 2.0 | 1.0 |
| 40 | 3.5 | 1.8 |
| 50 | 6.0 | 3.0 |
| 63 | 10.0 | 5.0 |

## 耗气量计算

### 单行程耗气量

```
Q = (π × D² / 4 × 10⁻⁶) × (P + 0.1013) / 0.1013 × S × 10⁻³
```

- Q: 耗气量 (L/行程)
- D: 缸径 (mm)
- S: 行程 (mm)
- P: 工作压力 (MPa)

### 每分钟耗气量

```
Q_min = Q × N × 2
```

- N: 每分钟动作次数
- 因子2: 伸出+缩回两个行程

## 安全系数选取

### 按负载类型

| 负载类型 | 安全系数 | 说明 |
|----------|----------|------|
| 静态负载 | 1.2~1.5 | 夹紧、保持 |
| 稳态动态 | 1.5~2.0 | 匀速运动 |
| 冲击负载 | 2.0~4.0 | 挡停、快速夹取 |
| 疲劳工况 | 2.5~4.0 | 高频次往复 |

### 按精度要求

| 精度等级 | 安全系数 | 说明 |
|----------|----------|------|
| 普通精度 | 1.2~1.5 | ±1mm |
| 中等精度 | 1.5~2.0 | ±0.5mm |
| 高精度 | 2.0~3.0 | ±0.1mm |

## 常用常数

- 重力加速度：g = 9.81 m/s²
- 标准大气压：P0 = 0.1013 MPa
- 常用工作压力：P = 0.5 MPa (0.4~0.6 MPa)
- 空气密度（标准状态）：ρ = 1.293 kg/m³
